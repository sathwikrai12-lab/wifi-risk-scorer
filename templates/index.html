How WifiGuard works
1. IP & Location Detection
Your IP is read server-side using a chain of free APIs (ip-api.com → ipwho.is → ipapi.co). We detect city, country, timezone, and ISP.
2. VPN / Proxy Detection
We check API proxy/hosting flags plus org name keywords (VPN providers, data center names) to estimate if your real IP is masked.
3. Risk Scoring (0–100)
Score combines: HTTPS (25pts) + No VPN (20pts) + ISP type (20pts) + Country risk (15pts) + Datacenter routing (20pts). Higher = more risk.
4. History & Privacy
Scan history is stored locally in your browser only. Nothing is ever sent to or stored on our servers. Each page load starts fresh on the backend.
5. Grades
A+ = 0–15 pts (very safe) · B = 16–30 · C = 31–50 · D = 51–70 · F = 71–100 (high risk)
